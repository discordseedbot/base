    LaunchParameters

    If entry.js is launched as "node core/entry.js --build" then the property "build" will be true. If entry.js is launched as "node core/entry.js --debug=false" then the property "debug" will be false. In other words you can force the state of a parameter by sort of treating it like a variable.
    Properties:
    Name Type Description
    build boolean Build Mode
    safe boolean Safe Mode
    inspect boolean Using Chrome Devtools Inspect
    debug boolean Debug Mode
    raw Object Raw Parameters, if it is not any of the supported ones they belong here.
